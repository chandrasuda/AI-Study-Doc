### Core Textbooks
1. **"Artificial Intelligence: A Modern Approach"** by Stuart Russell & Peter Norvig
   - Comprehensive introduction to AI concepts
   - Covers search, knowledge representation, learning

2. **"Pattern Recognition and Machine Learning"** by Christopher Bishop
   - Mathematical foundation for ML
   - Bayesian perspective on learning

3. **"Deep Learning"** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
   - Definitive guide to deep learning

### Path 1: Machine Learning Engineer (6-8 months)
1. **Month 1-2**: Math foundations + Python/pandas
2. **Month 3-4**: Classical ML algorithms + scikit-learn
3. **Month 5-6**: Deep learning + PyTorch/TensorFlow
4. **Month 7-8**: MLOps + deployment + portfolio projects

### Path 2: Research Scientist (8-12 months)
1. **Month 1-3**: Strong mathematical foundations
2. **Month 4-6**: Deep learning theory + paper reading
3. **Month 7-9**: Specialized area (NLP/CV/RL)
4. **Month 10-12**: Research project + paper writing

### Path 3: AI Product Manager (4-6 months)
1. **Month 1-2**: AI/ML concepts overview
2. **Month 3-4**: Industry applications + case studies
3. **Month 5-6**: Ethics, deployment, business impact
